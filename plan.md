#projeto Landing page para App simples
Página com fundo branco e fonte Roboto como principal.
O objetivo é ter uma landing page clean, branca e sofisticada.
HEADER
Logo no canto esquerdo
-A logo é um F com um círculo em volta (Feito em CSS)
-O fundo do círculo é verde Escuro

botão de download no canto direito
- fundo verde, cor branca
- borda arredondada

HERO SECTION
A hero deve ter altura de 90vh, dividido em 3 áreas horizontais
Esquerda:
Apenas a headline, sem ser grande demais, sem destaque (o destaque vai ser o print do app no meio).
"O app feito para acompanhar a sua vida fitness."

Centro:
Em destaque a imagem images/print_1.png que é o print do app

Direita:
Uma lista de pontos-chave sobro o app
- ❤ Saúde
- 🏃‍♂️ Exercício
- 🤝Amigos

SEÇÃO DE FUNCIONALIDADES
Vão ser usadas as imagens images/print_2.png, images/print_3.png e images/print_4.png que serão exibidas da seguinte forma
- print / texto
- texto / print
- print / texto

Ou seja, irá alternar entre o print na esquerda e um texto explicativo na direireita, vice-versa.

Descrição de cada print para você gerar um texto adequado
- images/print_2.png = Contagem de calorias
- images/print_3.png = Exercícios específicos (corrida,musculação,etc)
- images/print_4.png = Compartilhamento de metas com amigos
